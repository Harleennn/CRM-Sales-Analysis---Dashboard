# CRM Sales Analysis Dashboard 📊

An interactive 4-page CRM Sales Analysis Dashboard built using Power BI to explore sales performance, KPI tracking, customer insights, and AI-powered visual storytelling.

This project represents my personal learning journey in dashboard development, business analytics, and exploring AI visuals inside Power BI.


# Project Overview

The dashboard was designed to provide a unified reporting experience for analyzing:

- Sales performance
- Revenue trends
- KPI tracking
- Customer insights
- Regional analysis
- Interactive business reporting

The project focuses not only on analytics but also on improving dashboard usability and storytelling through navigation and interactivity.


# Key Features

## Interactive Dashboard Navigation
- Bookmarks
- Navigation buttons
- Multi-page unified dashboard experience

## Business Analytics
- Revenue analysis
- KPI monitoring
- Customer insights
- Sales trend analysis
- Region-wise performance tracking

## AI Exploration in Power BI
This project also explores AI-powered visuals in Power BI to understand how AI can enhance:
- Data storytelling
- Pattern recognition
- Insight generation
- User interaction
